Recipe Haven - Sharing Platform

Project Overview

A web-based recipe sharing platform where users can create, share, and discover recipes easily. Recipe Haven is designed to offer a clean interface, smooth navigation, and an engaging user experience for food enthusiasts.

Technologies Used

  • 🔹 HTML5 & CSS3
  • 🔹 JavaScript (ES6+)
  • 🔹 Node.js & Express.js
  • 🔹 MongoDB

Key Features

  • 🔐 User authentication
  • 📝 Recipe submission
  • 🔍 Search & Filters
  • ⭐ Rating & Reviews

What I Learned

Improved understanding of web development concepts, user authentication, and RESTful API design.
